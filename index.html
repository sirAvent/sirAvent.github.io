Wordi Bot

  • Python
  • discord.py
  • Flask

A fun bot I worked on that brings a game of Wordle to Discord servers. I recreated the game in the bot's code and made a website to showcase it. The bot is up 24/7 and is currently deployed in 7+ servers with over 65 users. Built with Python and discord.py and hosted on Replit HTTP servers using Flask.

  • CSS

A game where you play as a pig with a jetpack, flying to avoid obstacles in your way. Avoid as long as you can to gain the highest score. Core gameplay is inspired by Flappy Bird. I drew the sprite myself drawing reference from art by OMGPOP. Built with Javascript, CSS, and HTML5 Canvas.

Chat With Friends!

  • Python

A messaging application as the project for ICS 32. Implemented a login verfication system and encrypted messages in JSON objects for security. The GUI was built with Tkinter.
